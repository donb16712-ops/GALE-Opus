# Experiments — Master Index

The central registry for all prototypes, tests, and exploratory builds in GALE.  
This file defines the structure, categories, and conventions for experimental work.

## Experiment Types
- **Prototypes** — Early implementations of new ideas  
- **Tests** — Validation, verification, and behavior checks  
- **Explorations** — Open‑ended investigations and concept trials  
- **Sandboxes** — Temporary or disposable workspaces  

## Purpose
This registry maintains:
- The canonical list of experiments  
- Naming and versioning conventions  
- Links to operators, cosmotics, and diagrams where relevant  
- A clean boundary between stable and experimental logic  

## Status
This file is the authoritative root for all experimental work in GALE.  
All new experiments must register here.
