# Little‑Sis — Master Index

The central registry for all assistant modules and micro‑operators in GALE.  
Little‑Sis provides helper logic, support functions, and lightweight operator extensions.

## Module Types
- **Micro‑Operators** — Small, focused operator units  
- **Helpers** — Support logic for operators and cosmotics  
- **Assistants** — Task‑specific modules that extend GALE’s capabilities  
- **Wrappers** — Interface layers and convenience utilities  

## Purpose
This registry maintains:
- The canonical list of Little‑Sis modules  
- Naming and versioning conventions  
- Routing rules for assistant logic  
- Links to operators and cosmotics where needed  

## Status
This file is the authoritative root for all Little‑Sis modules.  
All new assistant logic must register here.
