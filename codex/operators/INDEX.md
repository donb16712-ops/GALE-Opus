# GALE Operators — Master Registry

The central index for all operator modules in GALE.  
This file defines the operator categories, naming conventions, and routing structure.

## Operator Categories
- **Core Operators** — Fundamental system actions and primitives  
- **Transform Operators** — Conversions, mappings, and structural transforms  
- **Analysis Operators** — Pattern detection, evaluation, and interpretation  
- **Synthesis Operators** — Generative, constructive, and assembly logic  
- **Utility Operators** — Helpers, wrappers, and support functions  

## Purpose
This registry maintains:
- The canonical operator list  
- Naming and versioning rules  
- Routing conventions  
- Operator‑to‑chamber linkage  

## Status
This file is the authoritative root for all operator modules.  
All new operators must be registered here.
