# Drafts — Master Index

The central registry for all early writing, notes, and in‑progress documents in GALE.  
This file defines the categories, conventions, and routing structure for draft material.

## Draft Types
- **Notes** — Quick captures, ideas, and rough thinking  
- **Working Docs** — In‑progress writing and partial builds  
- **Outlines** — Structural plans and document skeletons  
- **Fragments** — Small pieces awaiting integration  

## Purpose
This registry maintains:
- The canonical list of draft materials  
- Naming and versioning conventions  
- Links to codex, operators, and cosmotics where relevant  
- A clean boundary between drafts and stable content  

## Status
This file is the authoritative root for all draft material in GALE.  
All new drafts must register here.
